Is 2. closer to 1. or 3.?
1. 70 45
2. 60 44
3. 52 40

Normalization:
x' = (x-x_min)/(x_max-x_min)
(60-52)/(70-52)=0.44
(44-40)/(45-40)=0.8

1. 1    1
2. 0.44 0.8
3. 0    0

2. is closer to 1.
